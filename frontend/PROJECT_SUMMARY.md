# 食知（FoodSense）- 智能饮食决策辅助系统

## 项目概述

食知（FoodSense）是一款Android移动端应用界面原型，定位为健康管理类工具型应用，强调"健康状态感知与饮食决策辅助"。系统不提供任何医疗诊断或治疗功能。

## 设计特点

### 整体风格
- **视觉风格**：简洁、克制、专业、非医疗化
- **色彩方案**：
  - 主色：绿色（#10b981）、蓝色（#3b82f6）
  - 警告色：橙色（#f59e0b）
  - 风险色：红色（#ef4444）
  - 背景：浅色为主（白色/浅灰）

### 信息呈现
- 卡片式布局
- 结构化组件（标签、图表、徽章）
- 清晰的信息层级
- 强调可解释性

## 核心页面

### 1. 首页（Home.tsx）
**功能**：
- 拍照识别入口（大型相机按钮）
- 即时分析结果展示
- 决策建议等级（推荐/谨慎/不推荐）
- 风险标签提示
- 快捷统计数据

**特色**：
- 视觉中心化的拍照按钮
- 颜色区分的建议等级
- 加载状态动画
- 系统定位说明

### 2. 菜品识别与营养结构分析（FoodAnalysis.tsx）
**功能**：
- 菜品图片展示
- 识别结果（名称、烹饪方式）
- 主要食材列表
- 营养结构可视化（饼图 + 进度条）
- 热量水平区间估计
- 过敏原提示
- 详细饮食决策建议

**特色**：
- 双重营养展示（饼图和进度条）
- 颜色编码的营养素
- 详细的风险因素说明
- 免责声明

### 3. 功能工具模块（Tools.tsx）
**功能**：
- 快捷统计卡片
- 功能入口九宫格：
  - 营养分析记录
  - 健康报告
  - 饮食风险记录
  - 用药与饮食提醒
  - 设备健康数据
  - 数据统计分析
  - 饮食计划
  - 设置

**特色**：
- 清晰的功能分类
- 图标化设计
- 渐变色卡片

### 4. 用户健康画像（HealthProfile.tsx）
**功能**：
- 交互式3D人体模型
  - 支持拖动旋转
  - 状态高亮显示
  - 健康问题标注
- 基础信息管理
- 健康目标设置
- 健康状态摘要
- 历史记录入口

**特色**：
- 创新的3D可视化
- 非医疗化的抽象表达
- 渐变和阴影效果
- 动态脉冲动画

### 5. 健康提醒与报告（HealthReport.tsx）
**功能**：
- 今日提醒列表
  - 用药提醒
  - 饮食注意
  - 设备测量
- 智能提醒设置
  - 语音提醒
  - 设备联动
- 周报/月报
  - 趋势图表
  - 核心指标
  - 总结性文字
- 数据导出与分享

**特色**：
- 多维度图表（折线图、面积图）
- 趋势分析可视化
- 开关式交互控件
- 阶段性总结

## 技术栈

- **框架**：React 18.3.1
- **样式**：Tailwind CSS 4.x
- **图表**：Recharts 2.15.2
- **图标**：Lucide React
- **UI组件**：Radix UI
- **构建工具**：Vite
